## pf input mouseup

Release a mouse button (optional coords)

```
pf input mouseup [flags]
```

### Options

```
      --button int   button number (default 1)
  -h, --help         help for mouseup
      --x int        x coordinate (optional) (default -1)
      --y int        y coordinate (optional) (default -1)
```
