## pf input mouseup

Release a mouse button (optional coords)

```
pf input mouseup [flags]
```

### Options

```
      --button int   button number (default 1)
  -h, --help         help for mouseup
      --x int        x coordinate (optional) (default -1)
      --y int        y coordinate (optional) (default -1)
```

### Options inherited from parent commands

```
      --max-x int32   input coordinate space width (default 1920)
      --max-y int32   input coordinate space height (default 1080)
      --nested        auto-detect and connect to a nested Wayland session in /tmp
```
