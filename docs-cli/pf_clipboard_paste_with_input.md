## pf clipboard paste-with-input

Paste clipboard contents and send input text to the focused application

```
pf clipboard paste-with-input <text> [flags]
```

### Options

```
  -h, --help   help for paste-with-input
```

### Options inherited from parent commands

```
      --max-x int32   input coordinate space width (default 1920)
      --max-y int32   input coordinate space height (default 1080)
      --nested        auto-detect and connect to a nested Wayland session in /tmp
```
